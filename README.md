# Loto Spa - Versión estática para Vercel

Esta carpeta contiene una versión estática del sitio diseñada para desplegarse en Vercel.

## Archivos
- `index.html` — página principal estática
- `css/style.css` — estilos personalizados
- `js/main.js` — lógica del formulario y apertura de correo
- `images/logo.svg` — logotipo del sitio

## Cómo desplegar en Vercel
1. Abre tu cuenta de Vercel.
2. Importa el repositorio completo.
3. Configura el proyecto para servir la carpeta raíz del repositorio.
4. Despliega.

## Cómo funciona
- El formulario es estático.
- Al enviar, se abre tu cliente de correo con un mensaje preformateado a `info@lotospa.com`.
- No requiere PHP ni backend.

## Notas
- El sitio está listo para Vercel y funciona como una landing page y solicitud de cita.
- Si deseas más funcionalidad, se puede conectar luego a un servicio de email o API.
