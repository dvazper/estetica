### 1. Abre XAMPP Control Panel
- Inicia **Apache** (botón Start)
- Inicia **MySQL** (botón Start)

### 2. Crea la BD
Copia esto en navegador:
```
http://localhost/estetica/config/init_db.php
```
Espera a que veas: ✓ Base de datos inicializada

### 3. ¡Abre tu sitio!
```
http://localhost/estetica/
```

## 🆘 Si algo no funciona

### Error "Conexión rechazada"
```
→ Verifica que MySQL esté GREEN en XAMPP
→ Recarga la página
```

### Error "Tabla no existe"
```
→ Abre: http://localhost/estetica/config/init_db.php
→ Espera el mensaje ✓
```

### "No cargan los horarios"
```
→ Elige un día de lunes a viernes
→ Elige una fecha en el futuro
→ Recarga el navegador
```
